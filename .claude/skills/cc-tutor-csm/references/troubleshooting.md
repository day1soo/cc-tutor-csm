# 트러블슈팅 가이드

---

## M1: 환경 세팅

| 증상 | 원인 | 해결 |
|------|------|------|
| Claude Code가 실행 안 됨 | 설치 누락 또는 PATH 미설정 | `npm install -g @anthropic-ai/claude-code` 재실행 |
| "인증 실패" 메시지 | Claude 계정 미로그인 | `claude login` 실행 |
| CLAUDE.md 수정이 반영 안 됨 | 파일 저장 후 새 대화 시작 필요 | 대화 종료 → 재시작 |

## M2: 파일 다루기

| 증상 | 원인 | 해결 |
|------|------|------|
| "파일을 찾을 수 없습니다" | 경로 오류 또는 폴더 위치 | `ls samples/` 로 파일 존재 확인 |
| CSV 읽기 시 한글 깨짐 | 인코딩 문제 | "UTF-8로 읽어줘" 추가 지시 |
| Plan 모드가 안 켜짐 | 명령어 오류 | "Plan 모드로 전환해줘" 또는 `/plan` 입력 |
| 리포트 숫자가 틀림 | 필터링 조건 누락 | "수강시간 0인 행 제거 후 다시 계산해줘" |

## M3: MCP + Slack

| 증상 | 원인 | 해결 |
|------|------|------|
| "authorization URL 안 나옴" | MCP 서버 설정 누락 | `/mcp` 입력해서 Slack MCP가 목록에 있는지 확인 |
| 브라우저에서 "앱 추가 권한 없음" | 회사 Slack 정책으로 사용자 앱 추가 제한 | 워크스페이스 관리자에게 요청 또는 퍼실리테이터 테스트 워크스페이스로 대체 |
| 콜백 URL 붙여넣었는데 "invalid" | URL 복사 누락 (code=... 뒷부분 잘림) | 브라우저 주소창 URL **전체**를 다시 복사 |
| 메시지 검색 결과 없음 | 채널 권한 없음 또는 날짜 범위 부족 | 채널 가입 상태 확인, 날짜 범위 넓혀서 재검색 |
| 잘못 발송될까 걱정 | 학습 모드 규칙 미인지 | CLAUDE.md의 "Slack MCP 읽기·draft만" 규칙 준수 — 실제 발송 금지 |

## M4: 스킬 제작

| 증상 | 원인 | 해결 |
|------|------|------|
| 스킬이 실행 안 됨 | SKILL.md 문법 오류 | frontmatter (---) 블록 확인, triggers 형식 확인 |
| 스킬 결과가 매번 다름 | 실행 단계가 모호함 | SKILL.md의 단계를 더 구체적으로 수정 |
| "스킬을 찾을 수 없습니다" | 파일 위치 오류 | `.claude/skills/스킬명/SKILL.md` 경로 확인 |
| 스킬이 CLAUDE.md를 안 읽음 | CLAUDE.md가 프로젝트 루트에 없음 | 프로젝트 최상위에 CLAUDE.md 위치 확인 |

## 공통

| 증상 | 해결 |
|------|------|
| Claude가 느리게 응답 | 큰 파일은 부분만 읽도록 "처음 100행만 읽어줘" |
| 대화가 꼬인 느낌 | "새 대화 시작" → `/cc-tutor-csm` 재실행 |
| 뭘 해야 할지 모르겠음 | "도움" 입력 → 도움말 메뉴 |
| 에러 메시지 이해 안 됨 | 에러 메시지를 그대로 복사해서 Claude에게 "이 에러가 뭐야?" 물어보기 |
