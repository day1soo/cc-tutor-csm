# CSM팀 Claude Code 학습 프로젝트

## 학습 모드 규칙
- 실습은 samples/ 또는 my-data/ 폴더의 데이터로 진행
- Gmail / Slack MCP 사용 시 **읽기·검색·draft 작성만 허용** (실제 메시지 발송 금지)

## CSM팀 운영 규칙

### 리포트 작성 규칙
- 월간 운영 레포트 3개 섹션: 수강 강의 현황, 수강생 현황, 운영 사항
- 수강시간 표기는 HH:MM:SS 형식
- 인기 강의/우수 수강생은 각각 Top 5
- 수강 비중(%) = 수강 중 인원 / 총 수강인원

### 기업 운영 분류 체계
- 상태: ongoing(운영 중) / closed(종료)
- 운영난이도: High / Medium / Low
- 플랜 유형: 올, 올-커스텀, 싱글, 싱글-접수, 법정의무교육

### 신규딜 세팅 필수 항목
1) 기업명 2) 수강기간 3) 과정포맷 4) 운영과정정보
5) 수료기준 6) 대상명단 7) 기업담당자 8) 독려발송여부 9) 기타특이사항
