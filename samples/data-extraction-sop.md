# 월간 리포트 데이터 추출 SOP (수작업)

> 현재 CSM팀이 매월 리포트를 만들 때 따르는 수작업 절차입니다.
> M2에서 이 과정을 Claude Code로 자동화합니다.

## 사전 준비
- rawdata 다운로드: 데이터 스튜디오에서 기간 설정 후 추출
- **주의: 기간을 반드시 설정** (예: 3월 리포트 → 2026.03.01 ~ 2026.03.31)
- 수강시간 0 또는 공란인 데이터는 모두 삭제 후 시작

## 1단계: 인기 카테고리
1. rawdata의 카테고리명 확인 및 정리
   - 프로그래밍/데이터사이언스/비즈니스/마케팅/파이낸스/크리에이티브/직무/자기계발/리더십/외국어
2. 삽입 → 피봇 테이블
3. 행 영역에 카테고리명 추가 (순서: 내림차순, 정렬 기준: COUNTA)
4. 값 영역에 카테고리명 추가 (요약 기준: COUNTA, 표시: 열의 %)

## 2단계: 인기 강의 (수강시간 기준)
1. 삽입 → 피봇 테이블
2. 행 영역에 강의명 추가 (순서: 내림차순, 정렬 기준: 기간 내 수강시간(ss)의 SUM)
3. 값 영역에 기간 내 수강시간(ss) 추가 (요약 기준: SUM)
4. 상위 5개 강의 선택
5. **초 단위 시간을 00시간 00분 00초 형식으로 변환**하여 리포트에 추가

## 3단계: 인기 강의 (수강인원 기준)
1. 삽입 → 피봇 테이블
2. 행 영역에 강의명 추가 (순서: 내림차순, 정렬 기준: 강의명의 COUNTA)
3. 값 영역에 강의명 추가 (요약 기준: COUNTA)
4. 상위 5개 강의 선택

## 4단계: 우수 수강생 (수강시간 기준)
1. 삽입 → 피봇 테이블
2. 행 영역에 email 추가 (순서: 내림차순, 정렬 기준: 기간 내 수강시간(ss)의 SUM, 합계 표시 체크)
3. 행 영역에 이름 추가 (합계 표시 체크 해제)
4. 행 영역에 직급 추가 (합계 표시 체크 해제)
5. 행 영역에 소속 부서/팀 추가 (합계 표시 체크 해제)
   - **주의: email을 제일 먼저 추가 (기준 항목)**
6. 값 영역에 기간 내 수강시간(ss) 추가 (요약 기준: SUM)
7. 상위 5명 선택
8. 초 단위 시간 변환 후 리포트에 추가

## 소요 시간
- 숙련자: 약 30분/기업
- 초보자: 약 1시간/기업
- 담당 기업이 20개면: **월 10~20시간**
